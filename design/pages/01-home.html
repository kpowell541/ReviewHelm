Layout Structure

Full-screen ScrollView with no tab bar — the home screen is a hub that branches into each flow.

Settings icon moved from bottom footer to top-right header for quicker access.

Logo icon sits beside the app name for brand reinforcement.

Quick Stats Row

New addition — three stat cards give an at-a-glance dashboard feel:

Green Sessions completed count

Violet Average confidence across all items

Amber Active knowledge gaps count

Pulls from useSessionStore and useConfidenceStore.

Mode Cards

Each mode gets an icon circle with a tinted background matching its semantic color:

Violet Review — primary action, brand color

Green Polish — positive/approval association

Amber Learn — attention/growth association

Violet light Gaps — tracks back to brand

Badges on Learn and Gaps show active gap count in amber. Arrow affordance on right.

Recent Sessions

Shows last 3 sessions with title, date, mode, and coverage percentage.

Status badges use the semantic colors:

Complete — green

In Progress — amber

Tapping a card resumes/views that session.
