ReviewHelm v1.0.0

Structure

8 sections in a scrollable view, shown across 3 phone frames:

Frame 1: AI Tutor (keys + model)

Frame 2: API Usage + Budget Controls

Frame 3: Review Settings, Caching, Export, Data

AI Tutor Section

Claude API Key: Encrypted in expo-secure-store. Masked token display. Required for all AI features.

Admin API Key: Optional. Enables "Sync Official Cost" button to pull real spend from Anthropic's Admin API via costApi.ts.

Model picker: Sonnet is marked "Preferred" (was "Default"). Pricing shown per model. Hint clarifies Opus is for complex subjects only.

API Usage Section

2×2 stat grid: Tokens, spend, today's calls, days active.

Budget bar: Visual progress bar showing monthly spend vs budget. Color shifts from green → amber → red as usage increases.

Cost by feature: Breakdown showing spend per feature (Tutor Chat, Comment Draft, Learn Mode) with proportional mini-bars. Uses role from AiRequestOptions to attribute costs.

Sync Official Cost: Amber button. Calls fetchMonthlyCostFromAdminApi(). Grayed if no admin key.

Budget Controls

Monthly Budget: USD input. Default $40. Saved to useUsageStore.

Alert Thresholds: Comma-separated percentages. Triggers Alert.alert() in _layout.tsx when crossed. Default: 70, 85, 95.

Hard Stop: Toggle. When on, sendTutorMessage() throws error if projected cost would exceed budget.

Auto-Downgrade: NEW. When on and usage exceeds 80% of budget, forces Sonnet even if Opus is selected. Prevents expensive calls near limit.

AI Cooldown: NEW. Self-imposed minimum delay between API calls to prevent rapid-fire usage. Off / 3s / 5s (default) / 10s.

Caching

Cache Tutor Responses: NEW. When on, identical questions about the same checklist item return the cached response from local storage instead of making a new API call. Cache keyed on itemId + question hash.

Base content (whatItMeans, howToVerify, etc.) is always offline — loaded from bundled JSON, never hits the API.

Per-Session Cost

Not shown here in Settings — displayed at the end of each session in the Session Summary screen (Screen 7). Shows total tokens and estimated cost for that session using estimateCost().

Review & Display

Anti-Bias Mode: Shuffles section order in Polish mode.

Text Size: Small / Medium / Large chip selector.

Code Block Theme: NEW. Dark (default) or Light syntax highlighting. Stored in usePreferencesStore.codeBlockTheme.

New Features Summary

1. Admin API Key input for official cost sync

2. Model pricing labels ($3/$15 vs $15/$75 per 1M)

3. Monthly budget bar with visual progress

4. Cost-by-feature breakdown (Tutor / Comment / Learn)

5. Auto-downgrade toggle (Sonnet near budget)

6. AI cooldown selector (Off / 3s / 5s / 10s)

7. Response caching toggle for tutor

8. Code block theme selector (Dark / Light)

9. "Sync Official Cost" button with admin key
