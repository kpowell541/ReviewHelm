Purpose

The core screen of the app. Users work through a checklist section-by-section, rating each item. Used identically for Review and Polish modes — only the loaded checklist differs.

Polish mode with anti-bias enabled shuffles section order using a seeded PRNG (consistent per session).

Header

Title: Stack name + "Code Review" or "Polish My PR".

Subtitle: Session number and start date.

Score pills: Three metrics updated in real-time:

Coverage — % of items with a verdict

Confidence — average confidence across rated items

Issues — count of "Attention" verdicts

Progress bar: Violet gradient, fills with coverage %.

Session Notes & Search

Session notes: Collapsible. Captures context for the session summary. Auto-saves on blur.

Search: Filters items by text match across all sections.

Severity chips: Toggle filters. At least one must stay active. Active chips show full color; inactive are dimmed.

Section Headers

Sticky section headers with collapse/expand toggle.

Chevron: ▼ expanded, ▶ collapsed.

Progress: "responded/total" in JetBrains Mono.

Collapsed sections show only the header row.

Checklist Item — Collapsed

Status dot: Green = responded, dark outline = pending. 10px diameter.

Severity badge: BLK (red), MAJ (amber), MIN (blue), NIT (gray).

Item text: 2-line clamp with ellipsis.

Chevron: ▶ collapsed, ▼ expanded. Signals tappability.

No action buttons on collapsed rows — keeps the list clean and scannable.

Tap anywhere on the row to expand/collapse.

Checklist Item — Expanded

Verdict buttons: Three options with haptic feedback:

✓ Good (green) — item looks correct in the PR

⚠ Attn (amber) — found an issue worth flagging

— N/A (gray) — not applicable to this PR

Confidence scale: 1-5 buttons. Default 3. Color progresses red → green. Hint text below shows label ("Not confident", "Somewhat uncertain", etc.).

Notes toggle: Expands a text input for per-item notes. Shows "(saved)" when notes exist.

Contextual Nudges

📚 Deep Dive (green): Always visible when expanded. Opens the Deep Dive screen to read base content and chat with the AI tutor.

🎓 AI Tutor nudge (green): Extra prompt when confidence ≤ 2. Also navigates to Deep Dive.

Confidence 1: "Let me teach you this from scratch"

Confidence 2: "Want a refresher on this concept?"

✍️ Comment Drafter nudge (amber): Appears when verdict is "Attention". Opens Comment Drafter (Screen 9) where you describe the issue and get an AI-drafted review comment with refinement options (shorter, more detail, softer tone).

Complete Session

Green button at the bottom. Triggers confirmation alert.

If coverage < 70%, the alert warns: "Only X% covered — are you sure?"

On confirm: locks all scores, records completion, navigates to Session Summary (Screen 7).

Shows "Session Completed" (disabled) if already completed.
